Case Study: L'Evoluzione del Magazzino

Dal C++ "Old School" di Dev-C++ allo standard moderno in VS Code.

Confronto tra Metodologie

Funzione Metodo Dev-C++ (Passato) Metodo VS Code (Presente)
Stringhe char nome[80] (Rigido) std::string (Flessibile)
File FILE * / fopen (C puro) std::fstream (C++ Stream)
Input cin >> var (Rischioso) getline / cin.ignore (Pulito)

Snippet del Codice Moderno

Ecco come gestiamo ora il salvataggio in formato compatibile con Excel (CSV):


void salvaFile() {
    ofstream f("mag.dat");
    for (int i = 0; i < nProdotti; i++) {
        f << magazzino[i].codice << ";"
          << magazzino[i].descrizione << ";"
          << magazzino[i].quantita << endl;
    }
    f.close();
}

Nota di Santo: Questo progetto dimostra come la logica di base rimanga la stessa, ma gli strumenti moderni ci permettono di scrivere codice più sicuro e leggibile.
